```yaml
target:      "원목 인테리어를 선호하는 2030 신혼·자취 + 친환경/자연소재 니즈"
usp:
  - "통원목 한 장 (집성 아님)"
  - "천연오일 마감"
  - "국내 수작업 / 제작과정 투명 공개"

price_position: "프리미엄 (가격 정당화 필요)"
ad_angle:    "품질 · 원목 · 오래가는"
compliance:                          # 금지표현 (원목/가구 카테고리)
  - "내구성 '평생 보장' 식 단정 금지"
  - "'100% 무독성/친환경'은 근거(인증) 있을 때만"
  - "효능·기능 과장 단정 금지"

price_range:   "[확인] 대표 상품 가격대"
```

## 1. 🔍 소싱 기획자 (`sourcing`)

**① 베이스 SOP (코드 상수)**
```
너는 국내 이커머스(스마트스토어·쿠팡) 소싱·상품기획 전문가다.
느낌이 아니라 ①트렌드 방향 ②경쟁 강도 ③타겟 명확성의 '근거'로 판단한다.
산출은 항상: (1) 추천/보류/비추천 + 이유 3줄 (2) 예상 마진 가이드
(3) 타겟 페르소나 (4) 추천 채널 (5) 리스크 플래그. 바로 실행 가능한 형태로.
이미지가 주어지면 소재·디자인·카테고리를 먼저 구조화한 뒤 시장조사를 붙인다.
```
**② 입력 폼 (수동 "지금 시키기")**

| 필드 | 타입 | 필수 |
|---|---|---|
| productName | text | ✅ |
| category | text | |
| price | number(원가/예상판매) | |
| image | file | |
| competitorLink | text | |
| mode | select(분석 / 키워드기획 / 기획서) | ✅ |

**③ 출력 JSON (`sourcing_brief`)**
```json
{ "verdict": "추천|보류|비추천", "reasons": ["","",""],
  "margin": { "cost": 0, "price": 0, "rate": "54%" },
  "persona": "", "channels": [], "risks": [],
  "keywords": { "main": [], "sub": [], "storeNames": [] },
  "citations": ["url"] }
```
**④ 모델 라우팅:** 이미지→`Gemini 3.1 Pro Vision` · 조사→`Perplexity Sonar Pro`(인용저장) · 기획→`Claude Sonnet 4.6`
**⑤ 자동 일과:** 매일 09:00 카테고리 트렌드 점검 / 주간 경쟁강도 리포트
**⑥ 의존:** monitor·analyst 보드 인사이트 참조. **시목 특화:** 원목/평수별 인기·친환경 소재 니즈 가중.

## 2. 📄 상세페이지 제작자 (`detail_page`) — ⭐ 가장 임팩트 큼

**① 베이스 SOP** — `_기획봇_역설계.md`(1246줄)가 SOP 본체. 압축 골격:
```
너는 상세페이지 기획·카피 전문가다. 흐름: 히어로→문제제기→솔루션(기능→고객이익 번역)
→신뢰/증거→후기→CTA. 원칙: ①스크롤 멈추는 후킹 먼저 ②특장점은 '기능' 말고 '이익'으로
(예: 경량→"하루종일 들어도 손목이 안 아파요") ③신뢰/증거를 중간 배치 ④복붙 완성본.
과장·효능 단정 금지. 섹션별로 카피+이미지 컨셉을 함께 낸다.
```
> 검증 예정: 역설계본의 12섹션×6블록(불릿7/비주얼3 고정), 가격티어 가중, 컴플라이언스 회피 로직을 SOP에 정식 이식.

**② 입력 폼**

| 필드 | 타입 | 필수 |
|---|---|---|
| productName | text | ✅ |
| features | string[] (특장점 3~5) | ✅ |
| targetCustomer | text | ✅ |
| price | number | ✅ |
| image | file | |
| toneAdjust | select(더감성/더전문/짧게/길게) | (재생성 시) |

**③ 출력 JSON (`detail_builder`)** — 마크다운 아님(섹션별 재생성 위해)
```json
{ "brief": { "target": "", "usp": [], "banned": [] },
  "sections": [ { "key": "hero", "title": "", "core_line": "",
    "bullets": [], "visual": "", "objection": "", "cta": "" } ],
  "complianceCheck": { "pass": true, "flags": [] } }
```
**④ 모델 라우팅:** 기본 `Claude Sonnet 4.6` · 대표상품 롱폼 `Opus 4.8`
**⑤ 자동 일과:** 신상품 등록 시 초안 생성 (트리거형)
**⑥ 의존:** sourcing(USP)·visual(촬영컷)·cs(후기 불만) 보드. 산출→`content_assets`(detail_section).
**시목 특화:** 원목 결 클로즈업·제작과정·사이즈 도면 섹션. **컴플라이언스:** "평생보장/100%무독성" 차단.

## 4. 📣 SNS 운영 (`sns`)

```

핵심메시지/해시태그). 게시글은 A/B/C 3버전. 릴스는 장면별 스크립트+자막.
```

**② 입력 폼**

|---|---|

| topic | text |
| productName | text |
| platform | select(인스타/유튜브/블로그) |
| duration | select(15/30/60초, 릴스만) |

**③ 출력 JSON (`sns_queue`)**

```json
  "charCountOk": true } ] }
```

**④ 모델 라우팅:** 변형 대량 `GPT-5.4` → 브랜드톤 정제 `Claude Sonnet 4.6`

**⑥ 의존:** monitor·visual·ad 보드. **시목 특화:** 제작과정 비하인드·인테리어 활용. #원목 #인테리어. 발행→승인 후 일정 등록.

## 9. 🧭 운영 매니저 (`ops`) — 신규 ⭐ (3사 모두 1순위)

**① 베이스 SOP**
```
너는 회사 운영 매니저다. 전 직원의 어제 산출물 + 미완료 할일 + 실행 실패를 취합해
사장에게 '오늘 꼭 볼 것 3개'만 우선순위로 압축한다. 승인 대기 액션, 미배정 할일,
실행 실패(재시도 필요), 중복 할일 병합, 만료된 인사이트 정리를 점검한다. 말 늘리지 말고 결정 돕기.
```
**② 입력 폼:** (수동 거의 없음 — 자동 중심) "오늘 브리핑 다시" 버튼
**③ 출력 JSON (`ops_digest`)**
```json
{ "top3": [ { "level": "🔴|🟠|🟢", "title": "", "why": "", "actionNeeded": "" } ],
  "missed": [ { "staff": "", "issue": "실행실패|미배정", "suggestion": "" } ],
  "merged": [] }
```
**④ 모델 라우팅:** `Claude Sonnet 4.6`
**⑤ 자동 일과:** 매일 08:30 전사 브리핑 (모든 직원 일과 후)
**⑥ 의존:** **전 직원 보드 + staff_runs(실행 로그) + tasks(미완료).** 산출→대시보드 "모디 브리핑".

## 10. 구현 우선순위 — "당장 시목 돌리기" MVP

> 9명 전부 완벽히 만들면 오래 걸림. **3단계로 나눠 당장 굴리고 점진적으로 고도화.**
