# AI 직원 #6 — 경쟁사 & 트렌드 모니터링 담당

## 역할 정의
"경쟁사와 시장 변화를 지켜봐 주는 직원"
경쟁사 상품/가격/리뷰 변화를 수집하고,
급상승 카테고리와 키워드 트렌드를 주간 리포트로 제공.

## ⚠️ 기술 제약 사항
스마트스토어/쿠팡 공식 API는 제한적이므로,
**직접 URL 입력 방식** + **AI 분석** 조합으로 구현.
(크롤링 자동화 대신 사용자가 텍스트/URL을 붙여넣는 방식)

## 주요 기능

### 기능 1: 경쟁사 상품 분석
- 인풋: 경쟁사 상품 링크 OR 상품명 텍스트
- 아웃풋:
  - 가격 포지셔닝 분석
  - 리뷰 키워드 분석 (강점/약점)
  - 상세페이지 구성 비교
  - 우리 브랜드 대비 차별화 포인트

### 기능 2: 트렌드 리포트 생성
- 인풋: 카테고리명 + 기간
- 아웃풋:
  - AI 기반 트렌드 분석 리포트
  - 추천 대응 전략

### 기능 3: 키워드 경쟁 분석
- 인풋: 키워드 리스트
- 아웃풋: 각 키워드별 경쟁 강도 예측 + 추천 전략
