서체(폰트) 선택 비밀 공식
📖 상세페이지 폰트 선택 가이드
디자인 입문자를 위한 서체 톤앤매너 완전 정복
🔖 들어가며
폰트는 상품의 첫인상입니다.
옷차림만 봐도 그 사람의 분위기가 느껴지듯,
폰트 하나만 바꿔도 같은 상품이 고급스러워 보이기도, 싸 보이기도 합니다.
상세페이지에서 폰트는 단순히 글자가 아닙니다.
브랜드의 목소리이자, 고객이 무의식적으로 신뢰를 느끼는 기준입니다.
이 전자책 하나면 어떤 상품이든 톤에 맞는 폰트를 바로 고를 수 있습니다.
📌 1장. 폰트가 왜 중요한가?
🧠 고객은 폰트로 상품을 판단한다
아래 두 상황을 상상해 보세요.
상황 A: 프리미엄 홍삼 선물세트인데 폰트가 만화 느낌의 둥글둥글한 서체
상황 B: 아이 장난감인데 폰트가 딱딱하고 날카로운 명조체
둘 다 뭔가 이상한 느낌이 드시죠?
폰트가 상품의 분위기와 맞지 않으면, 고객은 이유도 모른 채 신뢰를 잃고 이탈합니다.
💡 폰트의 3가지 역할
① 분위기 전달 — 고급스러움, 귀여움, 신뢰감을 만듦
② 가독성 확보 — 읽기 편해야 오래 스크롤함
③ 브랜드 일관성 — 모든 페이지가 같은 느낌이어야 신뢰가 쌓임
📌 2장. 폰트의 종류부터 알자
입문자가 꼭 알아야 할 폰트 유형은 딱 3가지입니다.
✅ ① 고딕체 (Sans-serif)
"깔끔하고 현대적인 직장인"
획에 꺾임이나 장식이 없는 단정한 서체입니다.
가장 많이 쓰이고, 가장 무난하게 어울립니다.
대표 폰트: Pretendard, Noto Sans KR, Nanum Square, Gmarket Sans
어울리는 상품: 전자기기, 생활용품, 건강식품, 뷰티, 식품
✅ ② 명조체 (Serif)
"격식 있는 정장 차림"
획 끝에 작은 장식(세리프)이 있는 서체입니다.
고급스럽고 클래식한 느낌을 줍니다.
대표 폰트: KoPub 바탕체, 나눔명조, 본명조
어울리는 상품: 프리미엄 식품, 한방/전통 제품, 선물세트, 고가 화장품
✅ ③ 손글씨/감성체
"친근하게 말 거는 동네 친구"
사람이 직접 쓴 듯한 자연스러운 서체입니다.
따뜻하고 감성적인 브랜드에 잘 어울립니다.
대표 폰트: 카페24 써라운드, 빙그레체, 교보 손글씨
어울리는 상품: 수제 식품, 아이 용품, 반려동물 용품, 플라워/인테리어

📌 3장. 카테고리별 폰트 추천
🍖 식품 / 먹거리
비유: 음식 맛집 간판처럼, 첫눈에 맛있어 보여야 합니다.
일반 식품 (과자, 음료, 간편식) → 고딕체 (Noto Sans KR, Nanum Square)
프리미엄/건강식품 (홍삼, 유기농, 선물세트) → 명조체 (KoPub바탕, 본명조)
수제/감성 식품 (수제잼, 떡, 쿠키) → 손글씨체 (카페24 써라운드)
✅ 꿀팁: 식품 상세페이지에서 가격 숫자는 반드시 숫자가 선명한 고딕체로 표기하세요. 명조체로 쓴 가격은 읽기 어렵습니다.
💄 뷰티 / 화장품
비유: 백화점 화장품 매장 vs 다이소 화장품 — 같은 제품도 패키지가 다릅니다.
매스 뷰티 (저가~중가) → 고딕체 (Pretendard, Gmarket Sans)
프리미엄/럭셔리 → 명조체 + 영문 세리프 혼합
자연/유기농 브랜드 → 가는 고딕체 (얇은 웨이트의 Pretendard Light)
✅ 꿀팁: 뷰티는 폰트 굵기(웨이트) 가 핵심입니다. 얇을수록 고급스럽고, 두꺼울수록 강조 느낌입니다.
👶 아이 / 유아 용품
비유: 어린이집 간판처럼 둥글둥글하고 안전해 보여야 합니다.
전체 폰트 → 둥근 고딕체 (나눔바른고딕, 둥근모꼴)
포인트 강조 → 손글씨체 (귀여운 느낌)
절대 피해야 할 것 → 날카로운 명조체, 딱딱한 세리프체
✅ 꿀팁: 아이 용품은 부모가 삽니다. "안전하고 믿을 수 있다" 는 느낌을 주는 깔끔하고 부드러운 고딕체가 구매 전환에 유리합니다.
🏠 인테리어 / 생활용품
비유: 인테리어 잡지처럼 깔끔하고 여백이 살아있어야 합니다.
모던/미니멀 → 가는 고딕체 (Pretendard Light, Thin)
내추럴/우드톤 → 손글씨체 + 가는 고딕 혼합
프리미엄 가구 → 명조체 또는 영문 세리프
✅ 꿀팁: 인테리어 상품은 폰트보다 여백이 더 중요합니다. 폰트 크기를 줄이고 여백을 넓히면 즉시 고급스러워집니다.
🐾 반려동물 용품
비유: 귀엽지만 진지한 수의사 느낌 — 귀여움과 신뢰를 동시에 줘야 합니다.
감성/귀여운 제품 → 둥근 고딕 + 손글씨 혼합
건강/의료 관련 제품 → 깔끔한 고딕체 (신뢰감 우선)
💊 건강기능식품 / 영양제
비유: 병원 처방전처럼 깔끔하고 전문적이어야 합니다.
전체 → 단정한 고딕체 (Noto Sans KR, Pretendard)
성분/효능 강조 → 굵은 고딕 (Bold)
피해야 할 것 → 손글씨체, 개성 강한 디스플레이 폰트
📌 4장. 폰트 조합의 황금 법칙
폰트는 2가지만 쓰세요. 3가지 이상은 산만해집니다.
✅ 황금 조합 공식
text제목(헤드라인)   : 굵고 강한 폰트 (Bold, ExtraBold)
본문(설명 텍스트) : 얇고 읽기 편한 폰트 (Regular, Medium)

📌 5장. 폰트 크기 & 모바일 가이드
상세페이지는 80% 이상이 모바일로 봅니다.
PC에서 예쁘게 만들어도 모바일에서 깨지면 소용없습니다.

💡 비유: 폰트 크기는 목소리 크기와 같습니다.
다 같은 크기로 쓰면 어디가 중요한지 모릅니다.
크고 → 작고 → 더 작게 계층을 만들어야 눈이 자연스럽게 따라갑니다.
✅ 행간 & 자간 설정
text타이틀 80pt → 행간 100pt / 자간 -20~-40
본문  28pt → 행간 42pt  / 자간 -10~-20
💡 비유: 행간은 줄과 줄 사이의 숨 쉬는 공간입니다.
너무 좁으면 답답하고, 너무 넓으면 흐름이 끊깁니다.
📌 6장. 절대 하지 말아야 할 폰트 실수 5가지
❌ 폰트 3개 이상 섞기 → 산만해서 신뢰감 하락
❌ 본문에 Bold체만 쓰기 → 전부 강조하면 아무것도 강조 안 됨
❌ 배경과 폰트 색이 비슷하기 → 가독성 최악 (예: 노란 배경 + 흰 글씨)
❌ 모바일에서 폰트 너무 작게 → 24pt 이하 본문은 읽기 포기
❌ 상품 분위기와 안 맞는 폰트 → 홍삼 선물세트에 귀여운 손글씨체
