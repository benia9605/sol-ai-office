# AI 직원 #5 — 광고 기획 담당

## 역할 정의
"광고 소재와 세팅값 초안을 만들어주는 직원"
네이버 성과형, 카카오 모먼트, 인스타그램 광고에 쓸
카피와 타겟 세팅 가이드를 자동 생성.

## 주요 기능

### 기능 1: 광고 카피 세트 생성
- 인풋: 상품명, 핵심 USP, 타겟, 예산 규모
- 아웃풋:
  - 헤드라인 5가지 (15자 이내)
  - 서브 카피 5가지 (30자 이내)
  - 상세 설명 3가지 (90자 이내)
  - 권장 CTA 버튼 텍스트

### 기능 2: 광고 타겟 세팅 가이드
- 인풋: 브랜드 + 상품 + 예산
- 아웃풋:
  - 네이버 성과형 키워드 리스트 (20개)
  - 인스타그램 타겟 오디언스 설정 가이드
  - 일예산 배분 추천

### 기능 3: A/B 테스트 소재 세트
- 인풋: 메인 소재 컨셉
- 아웃풋: 변수가 다른 3세트 소재 (감성형/기능형/가격형)

## 브랜드별 광고 전략

| 브랜드 | 광고 채널 | 핵심 소구점 |
|--------|---------|-----------|
| 운명랩 | 인스타그램 | 감성/선물/일상 |
| 쏠닝포인트 | 네이버 쇼핑, 카카오 | 성분/효과/가격 |
| 시목 | 네이버, 인스타그램 | 품질/원목/오래가는 |
