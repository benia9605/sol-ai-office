    ## 1) 현재 프롬프트 템플릿 구조(기본 골격)
    
    대부분 프롬프트는 아래 **6덩어리**로 구성돼 있어요.
    
    1. **주인공(Subject)**
    - `[Product Name]` / `A model` / `[Food Item]` 같은 “무엇을 찍는지”
    1. **행동/연출(Action / Scene)**
    - `floating`, `splashing`, `placed on...` 처럼 “무슨 상황인지”
    1. **배경/소품(Background / Props)**
    - `white marble pedestal`, `laboratory equipment blurred`, `mint leaves` 등
    1. **조명/분위기(Lighting / Mood)**
    - `soft sunlight`, `golden hour`, `dark moody lighting`, `clinical` 등
    1. **사진 품질/스타일(Quality / Style tags)**
    - `photorealistic`, `commercial photography`, `studio lighting`, `8k`, `sharp focus` 등
    1. **출력 옵션(Parameters)**
    - `-ar 4:5` (비율) 같은 마지막 옵션
    
    > 한 줄 공식으로 외우면:
    > 
    > 
    > **[무엇] + [연출] + [배경/소품] + [조명/무드] + [스타일/퀄리티] + [옵션]**
    > 
    
    ---
    
    ## 2) 수정할 때 이렇게만 하면 됨(초간단 가이드)
    
    ### A. “바꾸고 싶은 것”은 해당 덩어리만 교체
    
    - 제품 분위기 바꾸기 → **조명/무드**만 교체
    - 배경 바꾸기 → **배경/소품**만 교체
    - 더 고급스럽게 → **스타일/퀄리티**에 `luxury`, `editorial`, `premium` 계열 추가
    
    ### B. 수정 순서(실수 적게)
    
    1. **Subject** 먼저 확정 (제품명/카테고리)
    2. 그 다음 **배경/소품**
    3. 마지막에 **조명/무드**로 톤을 완성
    4. **스타일/퀄리티**는 “필요한 것만” 5~10개 유지
    5. 맨 끝 `-ar`만 원하는 비율로 변경
    
    ### C. 딱 3가지만 주의
    
    - **옵션은 항상 맨 끝**: `-ar 4:5` 같은 거
    - **키워드는 너무 많으면 오히려 흔들림**: 10~15개 내로
    - **충돌 키워드 같이 쓰지 않기**:
        - `dark moody` + `bright clean` 같이 상반되는 분위기 동시 사용 X
    
    ## 3) 바로 써먹는 “수정 템플릿(복붙용)”
    
    아래 틀에 원하는 단어만 갈아끼우면 끝.
    
    ```
    [Subject] [Action/Scene], [Background/Props], [Lighting/Mood], [Style/Quality], --ar 4:5
    
    ```
    
    예)
    
    ```
    [Product Name] floating in mid-air, soft bubbles, clean pastel background, soft diffused light, photorealistic, commercial photography, sharp focus --ar 4:5
    
    ```

### 📌 나노바나나 활용 필수 가이드

시작하기 전에, 이 프롬프트들을 나노바나나에서 200% 활용하는 방법입니다.

1. **대괄호 `[ ]` 필수 수정:** 템플릿 내의 `[Brand Name]`, `[Product Type]` 등 대괄호로 묶인 부분은 반드시 **구체적인 영문**으로 바꿔주세요. (예: `[Brand Name]` → `AESTURA`, `[Product Type]` → `Vitamin Serum`)
2. **비율 설정 (Aspect Ratio):** 프롬프트 끝에 비율 코드를 꼭 붙여주세요.
    - 인스타그램/상세페이지용: `-ar 4:5` (추천)
    - 웹사이트 배너용: `-ar 16:9`
    - 썸네일용: `-ar 1:1`
3. **부정 프롬프트 (Negative Prompt) 활용:** 나노바나나 설정에서 아래 단어들을 추가하면 퀄리티가 올라갑니다.
    
    > text, watermark, ugly, deformed hands, extra fingers, blurry, low quality, distorted
    > 
4. **본인 로고 및 상품 사진** 있다면 첨부하여 합성 목업 생성 해주세요.  **없다면 [  ]  텍스트**에 기입하여 만들면 됩니다. 
5. 아래 예제 사진들은 일관성을 보여주기 위함이니 본인 카테고리에 맞춰 세부적으로 수정하시면 됩니다.
6. **로고+패키지박스 두개 사진 첨부** 합성  또는 **제품사진+모델사진 첨부** 합성 또는 **제품사진+모델사진+배경사진** 3개 사진을 첨부하여 합성 /  프롬프트 대신 원하는 사진으로 합성하셔도 됩니다.
