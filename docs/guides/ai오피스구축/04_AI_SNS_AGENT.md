# AI 직원 #4 — SNS 운영 담당

## 역할 정의
"귀찮은 SNS 계획과 게시글 초안을 대신 써주는 직원"
월간 콘텐츠 캘린더 초안 + 개별 게시글 카피를 생성.
운명랩(인스타 감성), 쏠닝포인트(루틴 정보), 시목(라이프스타일)으로 구분.

## 브랜드별 SNS 특성

| 브랜드 | 주 채널 | 콘텐츠 유형 | 해시태그 전략 |
|--------|---------|-----------|-------------|
| 운명랩 | 인스타그램 | 감성 무드샷, 리뷰 공유, 기념일 | #인생네컷 #일상소품 계열 |
| 쏠닝포인트 | 인스타그램, 블로그 | 성분 카드뉴스, 루틴 인증, 전문가 팁 | #헬스 #보충제 계열 |
| 시목 | 인스타그램, 유튜브 | 제작과정 비하인드, 인테리어 활용 | #원목 #인테리어 계열 |

## 주요 기능

### 기능 1: 월간 콘텐츠 캘린더 생성
- 인풋: 월/년도, 프로모션 일정(선택)
- 아웃풋: 4주치 콘텐츠 캘린더
  - 각 게시물 날짜/요일
  - 콘텐츠 유형 (사진/카드뉴스/릴스/스토리)
  - 주제 및 핵심 메시지
  - 추천 해시태그

### 기능 2: 게시글 카피 생성
- 인풋: 주제, 상품명(선택), 이미지 설명(선택)
- 아웃풋:
  - 인스타그램 캡션 (첫 줄 훅 + 본문 + 해시태그)
  - 3가지 버전 A/B/C

### 기능 3: 스토리/릴스 스크립트
- 인풋: 주제 + 길이(15초/30초/60초)
- 아웃풋: 장면별 스크립트 + 자막 텍스트

## API 설계

```typescript
POST /api/agents/sns
{
  brandId: string,
  mode: 'calendar' | 'caption' | 'script',
  input: {
    month?: number,
    year?: number,
    promotionSchedule?: string,
    topic?: string,
    productName?: string,
    imageDescription?: string,
    duration?: 15 | 30 | 60
  }
}
```
