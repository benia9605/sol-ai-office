# 🏢 Sol AI Office - 기능 및 UI 설계서

**버전:** v1.0  
**최종 업데이트:** 2026-02-19

---

## 📌 개요

Sol AI Office는 1인 사업가 Sol님을 위한 AI 팀 가상 오피스입니다.

**핵심 컨셉:**
- AI 팀원들과 협업 (플래니, 마키, 데비, 서치)
- 모디(비서)가 전체 일정/기록/멘탈케어 관리
- 모든 기록이 체계적으로 쌓이고 연결됨

## 💛 모디 플로팅 버튼

### 위치
- 오른쪽 하단 고정
- 채널톡/카카오톡 상담 버튼처럼 항상 보임
- 다른 페이지로 이동해도 유지

### 디자인
```
┌─────────┐
│  💛     │  ← 노랑 동그라미 (모디 캐릭터 이미지)
│  모디   │  ← 작은 텍스트
└─────────┘

hover 시: 살짝 커짐 + 그림자 증가
클릭 시: 채팅창 슬라이드 업
```

### 클릭 시 채팅창

```
┌─────────────────────────────────────┐
│ 💛 모디 (비서)              [─] [X] │
├─────────────────────────────────────┤
│                                     │
│  안녕하세요 Sol님! 오늘 하루도      │
│  화이팅이에요! 💪                   │
│                                     │
│  오늘 할 일 정리해드릴까요?         │
│                                     │
│  [📅 일정 확인] [✅ 할일 보기]       │
│  [📚 독서 기록] [💭 그냥 대화]       │
│                                     │
├─────────────────────────────────────┤
│ [메시지 입력...]            [전송]  │
└─────────────────────────────────────┘
```

### 모디가 담당하는 것
```
📅 스케줄 관리 - 일정 추가/확인/리마인드
✅ 할일 관리 - 오늘 할 일 정리
💡 인사이트 정리 - 핵심 내용 기록
📚 독서/스터디 - 공부 기록
📝 일기/멘탈케어 - 고민 상담, 감사 일기
🎯 목표 점검 - 주간 리뷰
💬 잡담 - 그냥 대화, 아이디어 나누기
```

## 📂 메뉴별 상세 기능

### 📅 일정 (Schedules)

#### 기능
- 캘린더 뷰 (월간/주간/일간)
- 일정 추가/수정/삭제
- 컬러 태그로 종류 구분 (커스텀 가능)
- 루틴 반복 설정
- AI 대화에서 바로 추가 가능

#### 필드

| 필드 | 타입 | 설명 | 예시 |
|------|------|------|------|
| id | UUID | 고유 ID | - |
| title | TEXT | 일정 제목 | "와디즈 펀딩 오픈" |
| date | DATE | 날짜 | 2026-02-28 |
| time | TIME | 시간 (선택) | 14:00 |
| type | TEXT | 종류 (컬러) | "deadline" |
| project | TEXT | 프로젝트 | "운명랩" |
| repeat | TEXT | 반복 설정 | "weekly" |
| reminder | TEXT | 알림 | "1hour" |
| notes | TEXT | 비고 | "리워드 구성 확인" |
| conversation_id | UUID | 연결된 대화 | - |
| created_at | TIMESTAMP | 생성일 | - |

#### 종류 옵션 (기본값, 커스텀 가능)
```
🔴 마감 (deadline) - #EF4444
🟡 미팅 (meeting) - #F59E0B
🟢 개인 (personal) - #10B981
🔵 루틴 (routine) - #3B82F6
🟣 중요 (important) - #8B5CF6
⚪ 기타 (other) - #6B7280
```

### ✅ 할일 (Tasks)

#### 기능
- 리스트 뷰 / 칸반 뷰 선택
- 드래그로 순서/상태 변경
- 완료 체크 시 기록 유지
- 뽀모도로 타이머 내장
- 루틴 반복 설정
- AI 대화에서 바로 추가 가능

#### 필드

| 필드 | 타입 | 설명 | 예시 |
|------|------|------|------|
| id | UUID | 고유 ID | - |
| title | TEXT | 할일 제목 | "프롬프트 3번 수정" |
| type | TEXT | 종류 (컬러) | "urgent" |
| project | TEXT | 프로젝트 | "운명랩" |
| status | TEXT | 상태 | "todo" / "in_progress" / "done" |
| priority | TEXT | 우선순위 | "high" / "medium" / "low" |
| due_date | DATE | 마감일 | 2026-02-07 |
| estimated_time | INTEGER | 예상 시간 (분) | 120 |
| actual_time | INTEGER | 실제 시간 (분) | 107 |
| repeat | TEXT | 반복 설정 | "daily" |
| notes | TEXT | 메모 | "변수 통합 작업" |
| conversation_id | UUID | 연결된 대화 | - |
| completed_at | TIMESTAMP | 완료 시간 | - |
| created_at | TIMESTAMP | 생성일 | - |

#### 상태
```
⬜ 대기 (todo)
🔄 진행중 (in_progress)
✅ 완료 (done)
```

#### 종류 옵션 (기본값, 커스텀 가능)
```
🔴 긴급 (urgent) - #EF4444
🟡 중요 (important) - #F59E0B
🟢 일반 (normal) - #10B981
🔵 루틴 (routine) - #3B82F6
```

#### 뽀모도로 기능

│                                 │

### 📚 독서/스터디 (Readings)

#### 기능
- 책/강의별 카드 목록
- 진행률 트래킹
- 클릭 시 스터디 기록 타임라인
- AI 대화에서 스터디 기록 추가 가능

#### 필드 - 책/강의 (readings)

| 필드 | 타입 | 설명 | 예시 |
|------|------|------|------|
| id | UUID | 고유 ID | - |
| title | TEXT | 책/강의 제목 | "원씽" |
| type | TEXT | 종류 | "book" |
| author | TEXT | 저자/강사 | "게리 켈러" |
| status | TEXT | 상태 | "reading" |
| total_pages | INTEGER | 전체 페이지 | 300 |
| current_page | INTEGER | 현재 페이지 | 135 |
| progress | INTEGER | 진행률 (%) | 45 |
| rating | INTEGER | 평점 (1-5) | 4 |
| summary | TEXT | 한줄평 | "집중의 힘" |
| tags | TEXT[] | 태그 | ["생산성", "자기계발"] |
| link | TEXT | URL (강의면) | "https://..." |
| started_at | DATE | 시작일 | 2026-01-20 |
| completed_at | DATE | 완료일 | - |
| created_at | TIMESTAMP | 등록일 | - |

#### 필드 - 스터디 기록 (reading_logs)

| 필드 | 타입 | 설명 | 예시 |
|------|------|------|------|
| id | UUID | 고유 ID | - |
| reading_id | UUID | 책/강의 ID | - |
| date | DATE | 기록일 | 2026-01-24 |
| range | TEXT | 범위 | "3장 (p.45-67)" |
| duration | INTEGER | 읽은 시간 (분) | 45 |
| summary | TEXT | 핵심 내용 | "한 가지에 집중하면..." |
| quote | TEXT | 인상 깊은 문장 | "성공은 할 일 목록이..." |
| thoughts | TEXT | 내 생각 | "운명랩에 적용하면..." |
| action_items | TEXT | 실천할 것 | "가장 중요한 일 1개 정하기" |
| conversation_id | UUID | 연결된 대화 | - |
| created_at | TIMESTAMP | 생성일 | - |

#### 종류 옵션
```
📖 책 (book)
🎬 강의 (course)
📺 영상 시리즈 (video_series)
🎧 오디오북 (audiobook)
📄 아티클 시리즈 (article_series)
```

#### 상태 옵션
```
📖 읽는 중 (reading)
✅ 완료 (completed)
⏸️ 보류 (paused)
📋 읽을 예정 (planned)
```

### conversation_summaries SQL

```sql
CREATE TABLE conversation_summaries (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  room_id TEXT NOT NULL,           -- 'plani', 'maki', 'devi', 'searchi', 'modi'
  conversation_id UUID REFERENCES conversations(id),
  date DATE NOT NULL,
  summary TEXT NOT NULL,           -- 요약 본문
  key_points TEXT[],               -- 핵심 포인트 배열
  action_items JSONB,              -- [{task, status, due_date, assignee}]
  decisions TEXT[],                -- 결정 사항
  topics TEXT[],                   -- 논의 주제
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 방별, 날짜별 빠른 조회를 위한 인덱스
CREATE INDEX idx_summaries_room_date ON conversation_summaries(room_id, date DESC);
```
